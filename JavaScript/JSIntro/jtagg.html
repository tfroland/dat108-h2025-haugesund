JavaScript på tagg

Dette dokumentet demonstrerer hvordan JavaScript-kode kan legges på HTML-tagger.

En lenke som kjører Javscriptkode.

Lenken er lagt inn i dokumentet ved følgende kode:

<a href="javascript: window.alert('Hei alle sammen')">lenke</a>

Et klikk på dette p-element kjører Javascriptkode.

Avsnittet, p-elementet er lagt inn i dokumentet ved følgende HTML kode:

<p onclick="this.classList.toggle('jtagg_emph')">Et klikk på dette <strong>p</strong>-element kjører Javascriptkode.</p>

Hvis tid skal vi senere lære om metoden addEventListener. Vi bør bruke denne metoden nå vi skal knytte JavaScript-kode til hendelser på HTML-tagger, og ikke bruke attributter som onclick.

JavaScript i web-dokument

JavaScript er programkode som hentes fra webtjener og utføres på klient av nettleser. Også Java og Visual Basic har blitt benyttet som klientsidespråk, men JavaScript er mye mer benyttet. Visual Basic kunne kun benyttes av Internet Explorer, og Java krever at en plugin må installeres i nettleser.

Med JavaScript kan vi f.eks:

  • Forberede data som skal sendes fra form-skjema i nettleser til webtjener:
    • Sjekke at nødvendige felter er fylt inn,
    • sjekke at data er konsistente.
  • Kan gi bevegelse og dynamikk på webside:
    • Flytte/posisjonere tekst,
    • beregne størrelser og posisjoner til web-elementer,
    • skjule/vise tekst avhengig av omstendighetene,
    • forandre utseende, åpne nye vinduer etc.
  • Kan endre/lage/operer på HTML-strukturer og CSS i web-dokument og CSS-stilark:
    • Med f.eks. Ajax kan deler av websiden endres/oppdateres med nye data fra tjener uten at hele websiden må genereres på nytt.

JavaScript kode kan legges til HTML-dokument ved bruk av taggen script. Taggen puttes vanligvis på slutten av body elemententet. En diskusjon rundt plassering av script tag vil du finne i forelesningsnotatene.

JavaScript kode kan bakes inn i selve HTML-dokument, men kan også lastes fra egne filer. Med bruk av egne filer for JavaScript kan vi gjenbruke koe i forksjellige HTML-dokumenter. Dette kan også gi raskere tilgang til JavaScript koden hvis nettleser kan lste koden fra cache. Bruk av separate filer lar også nettleser laste JavaScript koden parallelt med lasting og byggin av HTML-dokumentet.

JavaScript i egne filer

Javscriptkode lastes fra egen fil med følgende kode i et HTML-dokument:

<script src="URL_eller_sti_til_javascriptfil"></script>

Attributtet src er URL til filen med JavaScript kode. Referanse til fil med Javacriptkode følger reglene som for href attributtet til elementet a. Dersom filen er plassert på samme tjener som webdokumentet angir vi stien fram enten som en relativ eller absolutt sti. Dersom JavaSript-filen er plassert på en annen webtjener angir vi url-en på normal måte som http(s)://webtjener/sti/javascriptfil. Flere JavaScript filer kan lasttes ved å benytte flere JavaScript tagger.

JavaScript kode innbakt i HTML-dokument

JavaScript kode kan bakes inn i HTML-koden. Koden under demonstrerer hvordan dette kan gjøres for et HTML5 dokument:

// JavaScript kode
</script>

Et HTML-dokument kan ha flere script tagger, og vi kan bruke både innbakt og og lastet fra filer i samme HTML-dokument.

JavaScript kode på tagg

JavaScript kode kan legges i lenker og direkte på knapper. Koden under demonstrerer dette:

<!-- JavaScript i lenke -->
<a href="javascript: void window.alert('Klikk meg')">En Javscriptlenke</a>

<!-- JavaScript kode på knapp -->
<button onclick="window.resizeTo(300,400)">Endre størrelse</button>
